#Library for Serial Debug Assistant from UWP based on STM32 HAL

#该库对应windows商店中的串口调试助手，可直接添加至CubeMX生成的工程中

>该串口调试助手可将变量数值绘制成波形图，具有刷新率高稳定性强等优点
>一般使用串口重定向printf语句实现发送，但printf语句转换数据格式耗时很长，无法用于高控制频率场景
>故作者开发此库，通过Serial_Debug()代替printf()，用于生成并通过DMA发送符合标准格式的数据包，方可满足高频场景

由于作者水平有限，库中难免存在一些不足和错误之处，诚请各位读者批评指正

格式如下：
波形通讯协议： 
名称 = 数值 + "\r\n" 。一行中包含多条数据可以用 ‘,’ 分割。
例：
添加两条曲线， 名称分别为 Line1 Line2
两种字符串格式
Line1=10.0 , Line2=200\r\n   以回车换行结束。
Line1=10.0 , Line2=200,   以逗号结束。
